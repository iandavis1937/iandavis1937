  5. Deployment                                                                                                              
                  
  Teacher one-time setup (~20 min):
  1. Create a Firebase project at console.firebase.google.com (free, just a Google account)
  2. Create a Realtime Database, copy the firebaseConfig object
  3. Paste the security rules (provided above)                                                                               
  4. Paste the config into index.html                                                                                      
                                                                                                                             
  Hosting the game (students need a public URL):                                                                             
  - GitHub Pages (recommended) — drag-and-drop upload, free, HTTPS

  Before each class session (~2 min):
  1. Pick a session ID (e.g. period3-mar25)
  2. Open dashboard.html?session=period3-mar25 on teacher's device
  3. Share index.html?session=period3-mar25 link with students via Google Classroom / chat / QR code
